# LinkKeep

URL shortener built with React and Node.js

## Demo
- API: https://linkkeep.vercel.app

## Features
- Shorten long URLs
- User login/register
- Link dashboard
- Click tracking

## Tech Stack
- React + Node.js
- MongoDB
- JWT Auth

1. Clone repo
```bash
git clone https://github.com/Shetty852/LinkKeep.git
